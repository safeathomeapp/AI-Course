# Session Notes

Use this folder to record notes for every working session and major
project milestone.

## Required Rule

- Add a new note (or update the active note) after each meaningful
  session.
- Do not close work without confirming session notes are captured.

## Naming Convention

Use: `YYYY-MM-DD_session-topic.md`

Example: `2026-03-06_curriculum-planning.md`

## Note Template

```md
# Session Note - YYYY-MM-DD - Topic

## Summary

What was completed in this session.

## Decisions

- Decision:
- Reason:

## Changes Made

- Files updated:
- Key additions/edits:

## Risks or Open Questions

- Risk:
- Mitigation or next action:

## Next Steps

1. 
2. 
```
