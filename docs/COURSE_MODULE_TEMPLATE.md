# COURSE_MODULE_TEMPLATE.md

## Purpose

Defines the structure used for every module.

Each module should take approximately 60--120 minutes.

## 1. Module Overview

Include: - Module Title - Duration - Learning Objectives - Prerequisites
(optional)

## 2. Concept Explanation

Explain: - why the concept matters - how AI interprets instructions -
common beginner mistakes

## 3. Demonstration

Show: - example prompt - AI output - explanation of why it worked

Include comparisons between poor prompts and structured prompts.

## 4. Guided Exercise

Provide: - task description - instructions - suggested prompt
structure - example solution

## 5. Real‑World Application

Explain how the concept improves real professional workflows.

## 6. Common Mistakes

Examples: - vague prompts - missing context - lack of output constraints

## 7. Portfolio Artefact

Learners produce something tangible such as: - prompt libraries -
workflow diagrams - structured reports

## 8. Reflection Questions

Encourage learners to apply the concept to their own work.

## 9. Optional Advanced Exploration

Provide deeper material for advanced learners.
