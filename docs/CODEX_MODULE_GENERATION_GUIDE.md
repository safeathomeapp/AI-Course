# CODEX_MODULE_GENERATION_GUIDE.md

Guide for AI tools generating course modules.

---

## Naming

Modules must follow:

T1M01
T2M01
T3M01

Example:

T1M01_introduction_to_ai.md

---

## Generation Steps

1 Read COURSE_INDEX.md
2 Read PROJECT_CONTEXT.md
3 Read COURSE_CURRICULUM_MAP.md
4 Use MODULE_SCAFFOLDER_TEMPLATE.md
5 Review docs/course_assets/ for reusable instructional assets.

---

## Rules

Modules must:

- remain platform neutral
- emphasise human accountability
- include demonstrations and exercises
