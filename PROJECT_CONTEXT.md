## Module Naming System

Modules use:

- `T1M01` - Tier 1 Module 1
- `T2M01` - Tier 2 Module 1
- `T3M01` - Tier 3 Module 1

Example:

`T1M01_introduction_to_ai.md`

This ensures correct sorting and avoids numbering conflicts.
